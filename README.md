## Description
QuoteWall is a web based platform where users can generate new quotes based on how they are feeling at that moment. AI will tell quotes based on the feeling along with the authour name and a brief description of the author. Users can save their favourite quotes.

## Tech Stack
Frontend:  HTML, CSS, JavaScript Backend: Python (Flask) OpenAI Api for generating quotes.

## Motivation
The motivation to build this project was two reasons:
First one is that in life, there are many moments where we are feeling of not doing, feeling of giving up, but a right insprational quote always helps. 
Second thing was I realized that a person like me, i will go to google or chat gpt or instagram, but there isnt any focused platform on quotes, even if it did, i didnt knew it. I want to build something where I and people like me can generate verified and famous quotes based on how we are feeling at that moment. Some place where we can save our favourite ones and go back and read that all. 

## how it works
Okay so this is a two page based platform. User will select a feeling, and click the new quote button. OpenAI nano model will instantly generates a new quote, there is a heart in the top of the box, by clicking it, user can save their favourite quotes. To view the saved quotes, they can head to favourites page where they can see all their saved quotes. If user want to delete any quote, they can delete it by pressing the white box beneath it. 
I want to explain two important things:
First one is that I have tried hard to make sure that the AI dont repeat the same quote, and it improved, but still it sometimes repeat quotes.
Second thing is to make sure to refresh the favourites page to see the saved quotes.

## AI vs Non AI
The frontend of both pages was written all by me, the transition and the background color, in these two things i took AI help but not completely. In the backend, most of the use of AI was debugging, code was written by me. I went through lot of debugging and the devlogs explains that process in detail.
